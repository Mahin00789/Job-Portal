# Job Portal (MERN Stack)

A full-stack Job Portal application where recruiters can post jobs and manage applicants, and candidates can browse and apply for jobs.

---

## Features

### Authentication
- JWT-based login & signup
- Role-based access (Recruiter / Candidate)

### Recruiter
- Create job postings
- View jobs posted by them
- View applicants for each job
- Accept candidates

### Candidate
- Browse available jobs
- Apply for jobs
- View applied jobs

### Job Management
- Applicant tracking with status (`applied`, `accepted`)
- Secure protected routes

---

## Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** JWT
